Development of Minecraft multiplication machine

This machine is a binary circuit designed to take in 2 10-bit numbers, and output a 20-bit number after multiplication, all using entirely Minecraft redstone. It also can theoretically do the calculation instantly, without use of memory circuits.

Here is a test to demonstrate if the max value can be obtained:

Decimal: 1023 x 1023 = 1046529
